# The STK CLI
This command line tool is used to quickly generate empty projects.

To use the cli, run:
```
npm create stk

or

npx create-stk
```

To clone and run the cli, run:
```
git clone https://github.com/S-P-H-E/sphe-cli.git .
npm install

npm run dev
```

Supported Project Templates:
- Next JS
- Nuxt
- Svelte
- Node

Supported Package Managers:
- Npm
- Pnpm
- Bun
